使用套件时总是会好奇开发的流程是如何做到的？近期路过 Matt Pocock 的影片跟着做出一个套件发到 NPM 上，最棒的是很多步骤都达成自动化省下不少功夫：

1. 使用 tsup 将 ts 转为 esm 与 cjs 模块格式
2. 使用 changeset 管理更新 changelog 与 version
3. 使用 GitHub Actions 自动化 CI/CD 流程
